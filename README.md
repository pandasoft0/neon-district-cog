# Installation

```
git clone https://github.com/NeonDistrict/neon-district-cog.git
cd neon-district-cog
npm install
```

## Cog Software

```
cd server
pip install -r requirements.txt
```

# Use

In the root directory, just run with npm:

```
npm run dev
```

# Production Use

In the root directory, just run with npm:

```
npm run prod
```
